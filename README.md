# TaskFlow — Professional Task Manager

مشروع Portfolio احترافي مبني بلغات **HTML + CSS + JavaScript** فقط (بدون أي مكتبات JS خارجية).
يدعم **العربية والإنجليزية** مع تبديل RTL/LTR، ووضع داكن، وأيقونات **Font Awesome**.

## المميزات
- تسجيل دخول / إنشاء حساب مع تحقق فوري ورسائل خطأ ذكية
- كلمات المرور مُشفَّرة بـ SHA-256 عبر Web Crypto API قبل التخزين
- مؤشر قوة كلمة المرور + زر إظهار/إخفاء
- إدارة مهام: إضافة، تعديل، حذف، إكمال، بحث، فلترة، ترتيب، تصنيفات
- عدّاد إجمالي/مكتمل/متبقي + شريط تقدّم مباشر
- تبديل اللغة (AR/EN) والوضع (Light/Dark) في الشريط العلوي
- تصميم متجاوب بالكامل + أنيميشن خفيف
- تنبيهات Toast لكل عملية

## التشغيل
- **بدون خادم:** افتح `index.html` مباشرة في المتصفح.
- **مع خادم Node:** `node server.js` ثم افتح `http://localhost:3000`.

## الملفات
```
index.html   — الهيكل
style.css    — التصميم (Light/Dark + RTL)
script.js    — المنطق (Auth, i18n, Tasks)
server.js    — خادم Node اختياري
icon.svg     — شعار التطبيق
data/        — تخزين JSON للخادم
```
